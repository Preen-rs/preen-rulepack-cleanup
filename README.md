# preen-rulepack-cleanup

Official Preen capability rulepack for **Preen Cleanup Base**.

## Scope

- Pack ID: `preen-rs.cleanup.base`
- Smart Care capability: `cleanup`
- OS targets: `Macos`, `Linux`

## Local validation

```bash
python3 scripts/validate_pack.py
```

> Note: `manifest.sig` and `manifest.cert` are placeholders until the first release run.

## Release

Use workflow **Release Rulepack** and provide `x.x.x` version.
